• WYSIWYG: A type of editing software that allows users to see and edit content in a form that appears as it would when displayed on an interface, webpage, slide presentation, or printed document. WYSIWYG is an acronym for "what you see is what you get."

• HTML: HTML (Hypertext Markup Language) is a standard markup language used to structure and present content on the World Wide Web. It uses a set of tags to define the elements of a web page, such as headings, paragraphs, links, images, and more.

• CSS:(Cascading Style Sheets) is a stylesheet language used to define the visual presentation and layout of HTML elements on a web page, enabling designers to control aspects like fonts, colors, spacing, and responsive design.

• JavaScript: A versatile programming language commonly used for adding interactivity and dynamic behavior to web pages, allowing developers to create responsive and engaging user experiences.

Introduction to Mobile App Development

Mobile App Development: the creation and maintenance of applications tailored for mobile devices, focusing on design, coding, and optimization to deliver a user-friendly experience on smartphones and tablets.

To make iOS apps, I use Swift programming language and Xcode for the IDE i.e Integrated Development Environment, and to make Android apps, I use Java programming language and Android Studio for the IDE. They are all very popular and widely used 😁

What was used to make iOS apps before Swift and Xcode?

Before Swift, Objective-C was the primary programming language used for iOS app development for many years and is still used in some existing projects in conjunction with the Xcode IDE. Xcode is the official IDE for iOS, macOS, watchOS, and tvOS app development. The introduction of Swift by Apple in 2014 brought a more modern and approachable language to the iOS development ecosystem. Swift was designed to be safer, more concise, and offer improved performance compared to Objective-C.

What was used to make Android apps before Java and Android Studio?

Before Android Studio, Eclipse was the popular and widely used IDE for various programming languages, including Java and Kotlin. Android developers used Eclipse with the Android Development Tools (ADT) plugin to create Android applications. Android app development primarily relied on Java.
